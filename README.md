LifeLens AI

Overview

LifeLens AI is an AI-powered healthcare monitoring platform optimized for Arm-based edge devices.

Features

- Real-time vital sign monitoring
- Early health risk prediction
- Edge AI processing
- Low latency alerts
- Privacy-focused architecture

Tech Stack

- Python
- TensorFlow Lite
- Flask
- Raspberry Pi
- SQLite
- Arm Processors

Installation

pip install -r requirements.txt
python app.py

Future Scope

- IoT integration
- Telemedicine support
- Advanced disease prediction
